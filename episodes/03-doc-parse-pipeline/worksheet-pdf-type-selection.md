# 资产 1 — PDF 4 档选型工作表

> 来源：费曼学 AI 造物车间 #03 · 实测 200+ 尽调文档

## 使用方法

拿到一份 PDF 时，**按这个工作表 5 个问题逐个回答**，最后一行就是推荐方案。

---

## 工作表（直接打印 / 复制到 Notion 即可）

### 问题 1：你的文档类型？

| 选项 | 特征 | 继续 |
|---|---|---|
| **数字 PDF** | 可在 PDF 阅读器里选中/复制文字 | → Q2 |
| **扫描件** | 打开只能看图，不能选文字 | → **L3 档 4 (MinerU)** |
| **混合** | 部分页数字 + 部分页扫描 | → **L3 档 3 (PyMuPDF) + L4 OCR 降级** |

### 问题 2：你的核心需求是什么？

| 选项 | 推荐档 | 时延 | 成本 |
|---|---|---|---|
| **A. 全文检索** | L3 档 1 (PDF-dots) | 0.5s/页 | 0 |
| **B. 保留表格位置** | L3 档 2 (PDFPlumber) | 1-2s/页 | 0 |
| **C. 复杂排版（多栏/旋转/嵌入图）** | L3 档 3 (PyMuPDF) | 2-3s/页 | 0 |
| **D. 扫描件 OCR / 复杂版式终极** | L3 档 4 (MinerU) | 10s/页 | 0 |

### 问题 3：你的文档量级？

| 量级 | 建议 |
|---|---|
| **< 50 页/天** | L3 档 1（极简，能用就行） |
| **50-500 页/天** | L3 档 2（带表格） |
| **500-5000 页/天** | L3 档 3（混合排版） |
| **> 5000 页/天** | L3 档 4 抽样 + L3 档 2 全量 |

### 问题 4：你的延迟预算？

| 预算 | 推荐 |
|---|---|
| **< 1s/页** | L3 档 1 |
| **1-3s/页** | L3 档 2 / 档 3 |
| **可接受 10s/页** | L3 档 4 |
| **无所谓** | L3 档 4（最稳） |

### 问题 5：你的成本预算？

| 预算 | 推荐 |
|---|---|
| **0 元（必须免费）** | 档 1-3 全免费 |
| **$0.01-0.03/页** | 档 4 + VLM 降级 |
| **$0.10/页** | 全部走 VLM 兜底 |

## 实战组合（私募尽调项目）

我们用的实际方案：

```python
# 默认走档 2，扫描件走档 4
def route(pdf_path):
    if is_scan(pdf_path):
        return "mineru"
    return "pdfplumber"
```

| 文件类型 | 占比 | 走档 |
|---|---|---|
| 招股书数字 PDF | 40% | 档 2（PDFPlumber，1.5s/页） |
| 审计报告扫描件 | 30% | 档 4（MinerU，10s/页） |
| 行业研究报告 | 20% | 档 2 + OCR 兜底 |
| 复杂版式财报 | 10% | 档 3（PyMuPDF，3s/页） |

**总计**：平均 3.5s/页 × 200 文档 × 50 页 = **约 10 小时**。如果全走档 4 = 28 小时。

## 4 档代码骨架（每档 1 行启动）

```python
# 档 1: PDF-dots
from pypdf import PdfReader
text = "\n".join(p.extract_text() for p in PdfReader("x.pdf").pages)

# 档 2: PDFPlumber
import pdfplumber
with pdfplumber.open("x.pdf") as pdf:
    for page in pdf.pages:
        text = page.extract_text()
        tables = page.extract_tables()  # 关键：拿表格

# 档 3: PyMuPDF
import fitz
doc = fitz.open("x.pdf")
for page in doc:
    blocks = page.get_text("dict")["blocks"]  # 关键：拿 bbox
    tabs = page.find_tables()  # 关键：拿 DataFrame

# 档 4: MinerU
from magic_pdf.pipe.UNIPipe import UNIPipe
pipe = UNIPipe("x.pdf", model_dir="/models/MinerU")
pipe.pipe_classify(); pipe.pipe_analyze(); pipe.pipe_parse()
```
